# Google Page Clone

Welcome to the Google Page Clone repository! This project is a responsive clone of the Google search homepage, created using HTML and CSS. It closely mimics the design and layout of the original Google homepage, while being fully responsive for different screen sizes.

## Features

- Responsive design that adapts to different screen sizes
- Clean and minimalist layout, similar to the original Google homepage
- Fully functional search bar (design only, no backend functionality)

## Demo

[Live](https://mullaivenese03.github.io/Google-page-clone/)

## Getting Started

To use the Google Page Clone, follow these steps:

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/MullaiVenese03/Google-Page-Clone.git
   ```
2. Open the `index.html` file in your web browser to launch the Stopwatch application.

## Usage

1. Open the project in your web browser to see the Google homepage layout.
2. The design includes a search bar, buttons, and footer links, all styled to match the original Google page.
